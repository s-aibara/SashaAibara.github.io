Data Cleaning in SQL: Nashville Housing

SQL (PostgreSQL)

Goal:
The purpose of this project is to clean the dataset to make it more efficient for analysis.

Data:
For this project, I used raw data on the housing market in Nashville. The dataset is from Kaggle, and it includes the following variables: land use, property address, suite/condo, property city, sale date and sale price among others.

Techniques Used:
  • Corrected various structural errors
  • Handled missing data
  • Removed irrelevant data
  • Removed duplicates
