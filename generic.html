Data Exploration in SQL: Covid 19

SQL (PostgreSQL), Tableau

Goal:
This is an exploratory data analysis project where I examine global data on Covid 19. The main purpose of this project is to identify different patterns within the data, compare key statistics, and discover different relationships between variables. In this project, I identify the death and infection rates in different countries, compare the total number of Covid cases between countries and continents, and compare the vaccination rates in different countries.

Data:
The data for this project is obtained from Our World in Data. It includes data on variables grouped by country and continent such as the number of Covid cases, the number of Covid vaccinations, the number of hospitalizations, the population, the number of cases, and other relevant variables.

Key Insights:
  • In early 2021, the total number of Covid cases in the world was 150,574,977.
  • The total global deaths at this time was 3,180,206.
  • The continent with the highest death count in early 2021 was Europe.
